# üßÆ BMI-Calculator

A simple, user-friendly Body Mass Index (BMI) calculator application. This tool allows users to quickly calculate their BMI based on their height and weight inputs, providing immediate feedback on their weight status.

## ‚ú® Features

* **Simple Calculation:** Easily computes BMI using the standard formula:
    $$BMI = \frac{Weight (kg)}{Height (m)^2}$$
* **Weight Status Classification:** Provides the user with their corresponding weight category (e.g., Underweight, Normal, Overweight, Obese).
